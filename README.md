# 📝 TODO Website

A simple and interactive web app to manage daily tasks — add, delete, and mark tasks as completed.

## 🚀 Features

- Add new tasks
- Mark tasks as complete
- Delete tasks
- Clean, user-friendly interface

## 🛠️ Tech Stack

- HTML
- CSS
- Django
